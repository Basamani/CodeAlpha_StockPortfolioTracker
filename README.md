# 📊 Stock Portfolio Tracker (Python)

This is a simple Python project that helps users track their stock investments.  
It allows users to input stock names and quantities, calculates total investment value, and optionally saves the result to a file.

---

## 🚀 Features

- Track multiple stocks
- Predefined stock prices (AAPL, TSLA, GOOG, MSFT)
- Calculates total investment
- Displays individual stock value
- Option to save results to a file (`portfolio.txt`)
- Simple and beginner-friendly

---

## 🛠️ Technologies Used

- Python
- Basic concepts (loops, dictionary, file handling)

---

## ▶️ How to Run

```bash
python stock_portfolio.py
```

---

## 🧾 Example Input

```
Enter number of stocks: 2

Stock 1
Enter stock name: AAPL
Enter quantity: 2

Stock 2
Enter stock name: TSLA
Enter quantity: 1
```

## 📤 Example Output

```
AAPL price: ₹180
AAPL value: ₹360

Total Investment: ₹610
---------------------------
```

---

## 📁 Output File

If the user selects **yes**, the result will be saved in:

```
portfolio.txt
```

## ⚠️ Limitations

- Accepts only predefined stock names
